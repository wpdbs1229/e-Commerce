# 개요
간단한 커머스 프로젝트

Use : Spring, Jpa, MySQL, Redis, Docker, AWS

Goal : 셀러와 구매자 사이를 중개해 주는 커머스 서버를 구축한다.

## 회원
### 공통
- [ ] 회원가입
- [ ] 인중 ( 이메일 )
- [ ] 로그인 토큰 발행
- [ ] 로그인 토근 통한 제어 확인 (JWT, Filter를 사용)
- [ ] 예치금 관리

### 셀러
- [ ] 회원가입
